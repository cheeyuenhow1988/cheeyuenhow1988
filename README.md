## About

I like giving software a voice and a memory. Most of what I build is about agents that can actually do useful things: listen, keep context over time, and carry out tasks on their own. I'm self-taught in a lot of this and still learning in the open, so if something here is useful to you, that makes me happy.

## What I'm building

A few projects I'm working on. Most are private for now while they grow up, but here's the honest picture:

- **nova** 🔒 *(private)* — a voice-first personal AI assistant: cloud brain + laptop worker + phone PWA, all connected over a WebSocket bridge.
- **jimmy-brain** 🔒 *(private)* — the versioned "brain" (identity, rules, workflow, memory, skills) behind my always-on autonomous agent.
- **prompt-library** 🔒 *(private)* — the prompts I use to design and build AI agents.
- **claude-toolbox** 🔒 *(private)* — Claude Code skills and small infrastructure helper scripts I reach for often.
